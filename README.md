QuizApp

Um aplicativo de quiz interativo para a plataforma Android, desenvolvido utilizando as tecnologias modernas do ecossistema Kotlin e Jetpack Compose. O QuizApp oferece uma experiência de usuário fluida e engajadora, permitindo que os usuários testem seus conhecimentos em diversas categorias de perguntas, acompanhem seu progresso e compitam em um ranking global.

Funcionalidades

•Autenticação de Usuário: Registro e login de usuários via Firebase Authentication.

•Sistema de Quiz: Responda a perguntas de múltipla escolha com feedback instantâneo.

•Histórico de Quizzes: Visualize o histórico de quizzes realizados, incluindo pontuações e datas.

•Ranking Global: Compita com outros usuários e veja sua posição no ranking mundial, baseado nas melhores pontuações.

•Gerenciamento de Perguntas: Perguntas podem ser gerenciadas via Firebase Firestore, permitindo atualizações dinâmicas.

Tecnologias Utilizadas

O projeto é construído com as seguintes tecnologias e bibliotecas:

•Kotlin: Linguagem de programação principal.

•Jetpack Compose: Toolkit moderno para construção de UI nativa no Android.

•Firebase Firestore: Banco de dados NoSQL para armazenamento de perguntas e dados de ranking.

•Firebase Authentication: Gerenciamento de autenticação de usuários.

•Hilt (Dagger Hilt): Framework de injeção de dependência para Android.

•Coroutines e Flow: Para programação assíncrona e reativa.

•ViewModel: Gerenciamento de estado da UI de forma otimizada.

•Navigation Compose: Gerenciamento de navegação entre telas.
